# Mise en ligne Streamlit Cloud

1. Cree un nouveau repository GitHub propre.
2. Upload le CONTENU de ce dossier, pas le ZIP et pas le dossier parent.
3. A la racine du repository, tu dois voir directement :
   - streamlit_app.py
   - requirements.txt
   - claims_pilot/
   - data/
4. Dans Streamlit Cloud, choisis Python 3.12 et le fichier : streamlit_app.py

Ne selectionne jamais LANCER_CLAIMSPILOT_WINDOWS.bat dans Streamlit Cloud.
Ce fichier sert uniquement au lancement local Windows.
