# Déploiement Streamlit Cloud

1. Déposez le contenu de ce dossier à la racine du dépôt GitHub.
2. Dans Streamlit Cloud, sélectionnez : `streamlit_app.py`.
3. Dans les paramètres avancés, choisissez Python 3.12.
4. Après déploiement, vérifiez que la page affiche **ClaimsPilot V3.6.8**.

Cette version est stateless : elle ne réaffiche jamais une analyse précédente après changement de dossier.
